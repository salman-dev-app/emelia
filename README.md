```bash
# Clone the repository
git clone https://github.com/salman-dev-app/emelia.git
cd emelia

# Install dependencies
pip install -r requirements.txt

# Configure environment variables
cp .env.example .env
# Edit .env with your Bot Token, API keys etc.

# Run the bot
python main.py
```

> **Quick Deploy on Koyeb**: Fork repo → Connect to Koyeb → Add environment variables → Deploy 🚀
